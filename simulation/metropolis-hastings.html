Independence Sampler

Motivation: Note that we always travel closer to the mode if we can. What if we always travel closer to the mode but not quickly enough? Very slow progress that doesn't explore much of the posterior density.

For some density g(⋅)g(\cdot)g(⋅), J(θ∗∣θs−1)=g(θ∗)J(\theta^* | \theta^{s-1}) = g(\theta^*)J(θ∗∣θs−1)=g(θ∗) so that draws of θ∗\theta^*θ∗ are independent of θs−1\theta^{s-1}θs−1.

Example: J(θ∗∣θs−1)=N(θ∗;theta^,Σ^)J(\theta^* | \theta^{s-1}) = N(\theta^*; \hat{theta}, \hat{\Sigma})J(θ∗∣θs−1)=N(θ∗;theta^,Σ^)
